# AI Integrations

ZeroDust provides first-class integrations for AI agents and LLM frameworks.

## Available integrations

| Package | Framework | Install |
|---------|-----------|---------|
| `@zerodust/mcp-server` | Claude Desktop, Claude Code | `npx @zerodust/mcp-server` |
| `@zerodust/langchain` | LangChain.js | `npm install @zerodust/langchain` |
| `@zerodust/ai-sdk` | Vercel AI SDK | `npm install @zerodust/ai-sdk` |

## Tools provided

All integrations expose the same 6 tools:

| Tool | Description |
|------|-------------|
| `zerodust_info` | Service information and fees |
| `zerodust_get_chains` | List supported chains |
| `zerodust_get_balances` | Check balances across chains |
| `zerodust_get_quote` | Get a sweep quote |
| `zerodust_get_sweep_status` | Check sweep status |
| `zerodust_list_sweeps` | List past sweeps |

## Example prompts

Once configured, you can ask your AI:

- "What chains does ZeroDust support?"
- "Check my balances on 0x1234..."
- "Get a quote to sweep my Arbitrum ETH to Base"
- "What's the status of my sweep?"
- "Show my sweep history"
