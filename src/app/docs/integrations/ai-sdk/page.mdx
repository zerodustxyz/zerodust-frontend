# Vercel AI SDK Tools

The `@zerodust/ai-sdk` package provides tools for the [Vercel AI SDK](https://sdk.vercel.ai).

## Installation

```bash
npm install @zerodust/ai-sdk ai zod
```

## Usage with `generateText`

```typescript
import { createZeroDustTools } from '@zerodust/ai-sdk';
import { generateText } from 'ai';
import { openai } from '@ai-sdk/openai';

const tools = createZeroDustTools({ environment: 'mainnet' });

const { text } = await generateText({
  model: openai('gpt-4o'),
  tools,
  prompt: 'Check balances for 0x1234...',
});
```

## Usage with `streamText`

```typescript
import { createZeroDustTools } from '@zerodust/ai-sdk';
import { streamText } from 'ai';
import { anthropic } from '@ai-sdk/anthropic';

const tools = createZeroDustTools({ environment: 'mainnet' });

const result = streamText({
  model: anthropic('claude-sonnet-4-5-20250929'),
  tools,
  prompt: 'What chains does ZeroDust support?',
});

for await (const chunk of result.textStream) {
  process.stdout.write(chunk);
}
```

## Tool results

Unlike the LangChain integration (which returns text), the AI SDK tools return structured objects:

```typescript
// zerodust_get_balances returns:
{
  sweepable: [
    { chainId: 42161, name: 'Arbitrum', nativeToken: 'ETH', balance: '0.0008' }
  ],
  tooSmall: [
    { chainId: 137, name: 'Polygon', nativeToken: 'POL', balance: '0.000001' }
  ]
}
```

## Configuration

```typescript
const tools = createZeroDustTools({
  environment: 'mainnet',  // or 'testnet'
  apiKey: 'your-api-key',  // optional
  timeout: 30000,
});
```
